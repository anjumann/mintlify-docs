## Directory Structure

```bash
src/
 ├── app/                # Next.js App Router pages
 │   ├── dashboard/      # Dashboard routes
 │   ├── auth/           # Authentication routes
 │   └── [other routes]/ # Other application routes
 │
 ├── api/                # API clients
 │   └── product.ts      # Product API client
 │
 ├── auth/               # Authentication logic
 │
 ├── components/         # Shared UI components
 │   ├── animate/        # Animation components
 │   ├── hook-form/      # Form components
 │   └── [others]/       # Other reusable components
 │
 ├── hooks/              # Custom React hooks
 │
 ├── lib/                # Utility libraries
 │
 ├── routes/             # Routing definitions
 │   └── paths.ts        # Route path definitions
 │
 ├── theme/              # Material UI theme
 │   ├── index.ts        # Theme export
 │   └── palette.ts      # Color palette
 │
 ├── types/              # TypeScript type definitions
 │
 └── utils/              # Utility functions
     ├── axios.ts        # Axios configuration
     └── [other utils]/  # Other utility functions
```
